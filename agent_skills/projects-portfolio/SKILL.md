# Ricky Chilcott's Project Portfolio

## Active

### Stoked (2025-present, co-founder)
- **What:** Peer-to-peer brand advocacy platform for DTC brands. Connects prospects with real product owners through private SMS/email conversations.
- **Tech:** Rails 8, SQLite (production), Turbo/Stimulus, Twilio for SMS.
- **Status:** Active, with paying customers. Bootstrapped.
- **URL:** https://get.stokedhq.com

### Causey (2022-present, CTO/Operator)
- **What:** Strategic planning software for nonprofits and mission-driven organizations. Spun out of Mission Met.
- **Tech:** Rails, Postgres, Stimulus.
- **Status:** Operating with a small team. Several solo nonprofits and consulting firms also use it.
- **URL:** https://www.causey.app

### Imedview (2025-present, Senior Software Engineer)
- **What:** Case management software for the claims and litigation industry.
- **Role:** Full-stack engineering on a Rails-based platform.

- **Status:** Active; the umbrella for Ricky's solo / contract work.
- **URL:** https://www.rakefire.io

## Patterns across the portfolio

- **SaaS at small scale.** Multiple products run by small teams. Bootstrapped or near-bootstrapped.
- **Rails / SQLite / Postgres stack.** Pragmatic, boring tech. Rarely chases new frameworks.
- **Operational simplicity.** Recent move from Sidekiq to simpler background processing in Causey is a representative decision.
- **Long-running relationships.** Rakefire since 2011, Causey/Mission Met since 2016. Compounds over time.
- **Mission-adjacent.** Causey serves nonprofits; Stoked serves DTC brands building real customer relationships. Both reject "growth-at-all-costs" framing.
